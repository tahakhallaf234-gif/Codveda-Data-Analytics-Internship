# 🧹 Task 1: Data Cleaning & Preprocessing

The goal of this project is to clean and preprocess the Iris dataset before performing any analysis or machine learning tasks.

---

- **Dataset:** Iris Dataset
- **Rows:** 150
- **Columns:** 5

- Sepal Length

- Verify the cleaned dataset.
